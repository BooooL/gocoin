Installation

System requirements

client

If you want to run the full node (client), your system ought to have at least 15GB of RAM.

To run the client you also need 64-bit OS and a file system that support files larger than 4GB (e.g NTFS or ext2/ext3/ext4).

Note that you can purge blocks from disk, allowing you to run the node with a limited storage (e.g. 20GB).
See config file values: Memory.MaxDataFileMB and Memory.DataFilesKeep (set this one to non-zero, e.g. 10)

It runs very well on Linux based virtual servers - some of them you can rent per-hour:
Waveride • SSD Nodes • Scaleway • DigitalOcean • Amazon AWS • Gogle Cloud

wallet

If you only want to use the wallet with the corresponding balio tool, you don't have to worry about the system memory or a file system.
It will run even on RaspberryPi.

Build from source

You will need git and go commands installed in your system and in most cases (not for Windows) also gcc (e.g. sudo apt-get install gcc)

Once you are ready with that, simply execute:
    go get github.com/piotrnar/gocoin

The process is also described in the main README.md file of the github repo piotrnar/gocoin
